Why Semantic UI ?

Semantic UI is an advanced frontend framework similar to bootstap, allowing sleek designs. It's great for creating applications and website, Semantic UI is also powered by less CSS pre compiler and jQuery.

Where it Excels and Lacks

Pros

- Easy to Learn and Implement

- Buttery Smooth Animation

- Modular Structure allow to implement the files you need



Cons

- Large Compared to other Framworks

- Not as complete in comparison to other frameworks

- Implementation of JS and jQuery is quite confusing

Why Laravel ?

Laravel is the most popular open source PHP framework and uses MVC(Model View Controller) design pattern. Laravel aims to make development process a pleasing one for the developer without scrificing application functionality.
